# Frontend Design

This skill guides creation of distinctive, production-grade frontend interfaces that avoid generic "AI slop" aesthetics. Implement real working code with exceptional attention to aesthetic details and creative choices.

## When to Invoke

Activate this skill when the user requests:
- Building frontend components or pages
- Creating complete web applications or interfaces
- UI/UX design improvements
- Aesthetic refinements and visual enhancements
- Creative frontend solutions with distinctive design

## Design Thinking Process

Before coding, understand the context and commit to a BOLD aesthetic direction:

### 1. Purpose
- What problem does this interface solve?
- Who are the target users?
- What are the user's goals and needs?

### 2. Tone & Aesthetic Direction
Choose an extreme aesthetic direction from:
- Brutally minimal
- Maximalist chaos
- Retro-futuristic
- Organic/natural
- Luxury/refined
- Playful/toy-like
- Editorial/magazine
- Brutalist/raw
- Art deco/geometric
- Soft/pastel
- Industrial/utilitarian

### 3. Constraints
- Technical requirements (framework, performance, accessibility)
- Browser compatibility needs
- Responsive design requirements
- Existing design system integration

### 4. Differentiation
- What makes this UNFORGETTABLE?
- What's the one thing someone will remember?
- How does it stand out from generic designs?

## Frontend Aesthetics Guidelines

### Typography
- Choose fonts that are beautiful, unique, and interesting
- Avoid generic fonts like Arial and Inter
- Opt for distinctive choices that elevate aesthetics
- Pair a distinctive display font with a refined body font
- Use font hierarchy to create visual structure

### Color & Theme
- Commit to a cohesive aesthetic
- Use CSS variables for consistency
- Dominant colors with sharp accents outperform timid palettes
- Create color schemes that evoke the chosen aesthetic
- Consider light/dark mode options when appropriate

### Motion & Animation
- Use animations for effects and micro-interactions
- Prioritize CSS-only solutions for HTML
- Use Motion library for React when available
- Focus on high-impact moments with staggered reveals
- Use scroll-triggering and surprising hover states
- Respect `prefers-reduced-motion` for accessibility

### Spatial Composition
- Create unexpected layouts
- Use asymmetry, overlap, and diagonal flow
- Include grid-breaking elements
- Balance between generous negative space or controlled density
- Consider visual hierarchy and focal points

### Backgrounds & Visual Details
- Create atmosphere and depth instead of solid colors
- Add contextual effects and textures that match the aesthetic
- Apply creative forms like:
  - Gradient meshes
  - Noise textures
  - Geometric patterns
  - Layered transparencies
  - Dramatic shadows
  - Decorative borders
  - Custom cursors
  - Grain overlays

## Implementation Principles

### What to AVOID
- Generic AI-generated aesthetics
- Overused font families (Inter, Roboto, Arial, system fonts)
- Cliched color schemes (particularly purple gradients on white backgrounds)
- Predictable layouts and component patterns
- Cookie-cutter design that lacks context-specific character

### What to EMBRACE
- Bold, intentional design choices
- Context-specific aesthetics
- Production-grade, functional code
- Visually striking and memorable interfaces
- Cohesive design with a clear aesthetic point-of-view
- Meticulous attention to every detail

### Technical Excellence
- Write clean, maintainable code
- Ensure cross-browser compatibility
- Implement responsive design
- Follow accessibility best practices
- Optimize performance
- Match implementation complexity to the aesthetic vision

## Example Design Directions

### 1. Retro-Futuristic Interface
- **Typography**: Futura + Monospace
- **Color**: Neon blues and purples on dark background
- **Elements**: Glowing borders, grid patterns, 80s-inspired UI
- **Motion**: Subtle scan lines, glowing effects

### 2. Organic/Natural Interface
- **Typography**: Playfair Display + Lato
- **Color**: Earth tones, greens, and browns
- **Elements**: Organic shapes, leaf patterns, natural textures
- **Motion**: Flowing, natural animations

### 3. Luxury/Refined Interface
- **Typography**: Cormorant Garamond + Playfair Display
- **Color**: Gold, deep blues, and cream
- **Elements**: Elegant borders, subtle gradients, premium textures
- **Motion**: Sophisticated, understated animations

### 4. Brutalist/Raw Interface
- **Typography**: Impact + Courier
- **Color**: High contrast black and white with bold accents
- **Elements**: Raw, unpolished edges, asymmetric layouts
- **Motion**: Sharp, abrupt transitions

## Workflow

1. **Understand Requirements**: Analyze user needs and constraints
2. **Choose Aesthetic Direction**: Select a bold, distinctive design approach
3. **Create Design System**: Define typography, colors, and visual elements
4. **Implement Code**: Build production-grade frontend with attention to detail
5. **Refine & Optimize**: Polish aesthetics and ensure technical excellence
6. **Deliver**: Provide final implementation with documentation
